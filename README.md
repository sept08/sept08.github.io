## 更换电脑重建开发环境
1.  克隆本仓库
2.  切换至hexo依次执行:
    1.  `npm install hexo` 安装hexo运行的必须库
    2.  `npm install`
    3.  `npm install hexo-deployer-git` 安装hexo的部属发布环境

## 常用基本操作
*   新建文章  `hexo new [layout] <title>`
*   生成静态  `hexo generate`
*   启动服务  `hexo server`
*   部属网站  `hexo deploy`
